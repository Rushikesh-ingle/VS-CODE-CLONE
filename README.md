# VS Code Clone

A simple Visual Studio Code (VS Code) inspired user interface built using only **HTML5** and **CSS3**. This project replicates the basic layout and appearance of the VS Code editor to practice frontend development and CSS layout techniques.

## 🚀 Live Demo

https://zesty-sable-aee589.netlify.app/

## ✨ Features

* VS Code-inspired user interface
* Sidebar navigation
* Explorer panel
* Editor workspace
* Top navigation bar
* Status bar
* Clean and responsive design
* Built entirely with HTML and CSS
* No JavaScript used

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome Icons

## 🔧 Installation

1. Clone the repository

```bash
git clone https://github.com/Rushikesh-ingle/VS-CODE-CLONE.git
```

2. Open the project folder

```bash
cd VS-CODE-CLONE
```

3. Open `index.html` in your browser
